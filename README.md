> Gun milan with 8 attributes

## Installation

```sh
$ npm install --save ashtakoot
```

## Usage

```js
var ashtakoot = require('ashtakoot');
var girl = { nakshatra: 9, moonsign: 2 }
var boy = { nakshatra: 12, moonsign: 7 }
ashtakoot(girl, boy);
```
